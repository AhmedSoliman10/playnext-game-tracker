# Code of Conduct

PlayNext should be a practical, respectful place to build.

## Expected Behavior

- Be clear, kind, and constructive.
- Assume good intent, but accept correction gracefully.
- Critique code and ideas, not people.
- Keep issues and pull requests focused on the project.
- Respect privacy and never post secrets or private user data.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic comments after maintainers ask you to stop.
- Submitting malicious code or intentionally unsafe instructions.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; reject pull requests; or block users when behavior harms the project or community.

If you need to report a problem, contact the repository owner through GitHub.
